# Amaze UI 简介

Amaze UI 是一个轻量级、模块化、专注于现代浏览器的跨屏前端框架。


Amaze UI 有以下几个特点：

## 语义化

Amaze UI 开发遵循语义化原则，意图通过类名（class）等信息直观传达元素的功能角色，同时关注结构、样式、行为分离，降低各部分的耦合程度，提高开发效率和可维护性。

## 移动优先，跨屏适配

使用 Amaze UI，你可以从移动设备开始开发网站，逐步在扩展的更大屏幕的设备上，专注于最重要的内容和交互，适应移动互联潮流。遵循 ”移动优先（Mobile First）“、”渐进增强（Progressive enhancement）“的理念，轻松创建跨屏适配的网页。

## 模块化，按需定制

Amaze UI 使用 LESS 编写样式，结构良好，易扩展，易维护；使用 Seajs 模块化开发、组织 JavaScript，自然、优雅。

## 专注于 HTML5

Amaze UI 基于轻量的 Zepto.js 开发，有效减少为兼容旧浏览器的臃肿代码；基于 CSS3 的交互效果，平滑、高效。Amaze UI 专注于现代浏览器，不再为过时的浏览器耗费资源，为更有价值的用户提高更好的体验。Amaze UI 会引入更多的 HTML5 技术，推进 HTML5 的普及应用。



Amaze UI 不仅仅是一个前端框架，通过 Amaze UI，我们还希望传达一个理念：放开纷扰，享受当下，远瞩未来。是的，勇敢地向那些老旧的浏览器告别，执 Amaze UI，拥抱 HTML5。这是我们对前端开发的态度，也是对生活的态度。
